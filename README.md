# 挂起记录 MVP

这是根据 `PRD-在线达标助手.md` 搭建的本地优先 MVP，软件名称为“挂起记录”。

## 已实现

- 今日星期与默认班次识别
- 在线/挂起/恢复/结束计时
- 在线进度圆环、补时倒计时、预计完成时间
- 挂起参考余量和超出提醒
- 今日时间轴
- 本周、本月、历史统计
- Excel 兼容格式导出
- 深色、浅色、跟随系统主题
- 自定义班次和每周班次绑定
- 每日班次 CSV 导入和模板下载
- 浏览器本地自动保存和跨天初始化
- 悬浮窗开关入口

每日班次 CSV 表头格式：

```text
日期,班次名称,开始时间,结束时间,目标在线分钟
2026-07-15,早班,08:30,17:00,450
2026-07-16,中班,11:00,20:00,450
```

导入入口位于“设置 -> 每日班次导入”。导入日期会覆盖该日期的预设班次；已经开始计时的历史记录不会被覆盖。

## 当前默认口径

- 每日目标在线时长：450 分钟
- 周一至周五：早班 08:30-17:00
- 周六、周日：休息日
- 班次外补时：默认开启，未达标时可继续补足
- 数据：浏览器本地存储

## 下一阶段

当前已完成 Windows 桌面封装基础：系统托盘、独立悬浮窗、隐藏到托盘、托盘退出菜单和“挂起记录”应用图标。

## 构建 Windows 安装包

```powershell
npm run build
cargo tauri build --bundles nsis
```

安装包输出到：

`src-tauri/target/release/bundle/nsis/挂起记录_0.1.0_x64-setup.exe`

本机已安装 Rust、Visual C++ Build Tools 和 Tauri CLI。当前数据仍使用前端本地存储；后续再迁移到 SQLite。
